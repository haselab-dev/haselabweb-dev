FIB Dataset

In this page, we describe public dataset Fish Image Bank (FIB) which we developed.

About

Fish Image Bank (FIB)データセットは，JST ACT-X「AI活用で挑む学問の革新と創成」採択プロジェクト「水産業のビッグデータ化に向けた汎用的な漁獲量認識基盤の開発」の研究の一部として，長谷川研究室が収集，アノテーションを行っています．


利用に際しては，以下の文章をよく確認してください．

内容に問題を見つけた方は，t-hase[at]u-fukui.ac.jpまでお知らせいただけると幸いです．

Data description

  • FIBは24魚種405枚の4K画像で構成される魚画像データセットです．
  • 2022年3月12日から2023年9月15日までの間に，福井県内の市場や卸売業者にて流通した魚を対象に撮影を行っています．
  • 背景は単色で統一され，なるべく環境が統一されるように照明を当てています．
  • 1枚の画像には必ず魚が1尾含まれるようにしています．
  • 一部不要な背景は手動で除去しています．
  • 同一個体が複数回撮影されているものを含みます．
  • ラベルは画像を目視で確認し，手動で付与されているため，誤りを含む可能性があります．
