FIB Dataset

In this page, we describe public dataset Fish Image Bank (FIB) which we developed.

About

Fish Image Bank (FIB) Dataset is constructed by Hasegawa Laboratory. This project is supported by JST ACT-X AI powered Research Innovation / Creation entitled Development of a robust catch-recognition platform for constructing big data in the fisheries industry.


Please read the following statement carefully before use.

If you find any problems with the contents, please let us know at t-hase[at]u-fukui.ac.jp

Data description

  • FIB is a fish image dataset consisting of 405 4K images of 24 fish species.
  • The images were taken of fish distributed at markets and wholesalers in Fukui Prefecture, Japan between March 12, 2022 and September 15, 2023.
  • The backgrounds were uniformly monochromatic and illuminated to create as uniform an environment as possible.
  • Only a single fish is in an image.
  • Some unnecessary backgrounds are manually removed.
  • Multiple images of the same fish may be included in a single image.
  • Although we checked carefully, Labels are manually assigned by visual inspection of the images and may contain errors.
