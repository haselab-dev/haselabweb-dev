VSAED

In this page, we describe public dataset VR-Self-annotation Emotion Dataset (VSAED) which we developed.

About

VR-Self-annotation Emotion Dataset (VSAED) is constructed by Hasegawa Laboratory. This project is supported by JSPS KAKENHI entitled Realisation of an foundation model for the democratisation of activity recognition AI.

Data description

  • VSAED is an internal speech emotion recognition dataset consisting of 1352 utterances categorized into 6 emotion classes.
  • The subjects are 28 healthy individuals in their twenties (22 males and 6 females).
  • The emotion classes are Angry, Fear, Disgust, Neutral, Sad, and Happy.
  • Recordings were conducted in December 2024 in Fukui Prefecture, where an experimental environment was set up.

Compared to typical speech emotion recognition datasets, it has the following characteristics:

  • It is a machine dialogue dataset in which the addressee was informed that it is a machine (speech dialogue agent).
  • It distinguishes between cases where the addressee is a machine capable of emotion recognition and those where it is not.
  • It records emotions that occurred naturally in a VR setting without instructing subjects to express specific emotions.
  • The ground truth labels are assigned based on the subjects' self-reports.
