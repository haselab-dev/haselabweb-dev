VSAED

In this page, we describe public dataset VR-Self-annotation Emotion Dataset (VSAED) which we developed.

About

VR-Self-annotation Emotion Dataset (VSAED)は，JSPS科学研究費助成事業 基盤研究(C) 採択プロジェクト「行動認識AIの民主化に向けた基盤モデルの実現」の研究の一部として，長谷川研究室が収集，アノテーションを行っています．利用に際しては，以下の文章をよく確認してください．内容に問題を見つけた方は，t-hase[at]u-fukui.ac.jpまでお知らせいただけると幸いです．

Data description

  • VSAEDは6感情クラス1352発話で構成される内的感情認識データセットです．
  • 被験者は健常な20代の28名（男性22名，女性6名）です．
  • 感情クラスは，怒り（Angry），恐れ（Fear），不快（Disgust），中立（Neutral），悲しい（Sad），幸せ（Happy）の6クラスです．
  • 2024年12月に福井県内で実験環境を整備して録音を行いました．

一般的な感情認識データセットと比べ，以下の特徴があります．

  • 発話先が機械（音声対話エージェント）であることを伝えた，機械対話データセットです．
  • 発話先が感情認識できる機械である場合，そうでない機械である場合を分けて計測しています．
  • 感情表出を指示せず，VRを用いて自然に生じた感情を記録しています．
  • 正解ラベルを本人の申告に基づいて付与しています．
